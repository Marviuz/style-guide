# @marviuz/typescript

This package provides a standardized TypeScript configuration for consistent project setups.

## Installation

```package-install
npm i -D @marviuz/typescript
```

## Usage

```jsonc:tsconfig.json
{
  "extends": "@marviuz/typescript"
  // ... Your other TypeScript options
}
```
