# @marviuz/prettier-config

This package provides a standardized Prettier configuration, including `prettier-plugin-packagejson` for formatting `package.json` files.

## Installation

```package-install
npm i -D @marviuz/prettier-config
```

## Usage

```js:prettier.config.mjs
import marviuzConfig from '@marviuz/prettier-config';

export default marviuzConfig;

// I haven't tried this yet, but this MIGHT work
export { default } from '@marviuz/prettier-config';
// or
export * from '@marviuz/prettier-config'
```

## Extending the configuration

If you need to customize the configuration, you can extend it by destructuring and adding your own options.

```js:prettier.config.mjs
import marviuzConfig from '@marviuz/prettier-config';

/** @type {import('prettier').Options} */
export default {
  ...marviuzConfig,
  plugins: [...marviuzConfig.plugins, 'prettier-plugin-tailwindcss'],
};
```
